# QuickSorter

Имплементация на алгоритъма QuickSort на C++ с използване на клас и обикновен масив.

## Как работи

QuickSort е алгоритъм от типа "раздели и владей". Избира се **pivot** елемент и масивът се разделя на две части — елементите по-малки от pivot отиват вляво, по-големите вдясно. Процесът се повтаря рекурсивно докато масивът не е сортиран.

**Средна сложност: O(n log n)**

## Функционалност

- Обектно-ориентирана имплементация с C++ клас
- Работи директно върху оригиналния масив (без копиране)
- Поддържа `cout << sorter` за лесно принтиране чрез `operator<<`

## Употреба

```cpp
int arr[] = {3, 5, 8, 10, 2, 1, 2};
int size = sizeof(arr) / sizeof(arr[0]);

QuickSorter q(arr, size);
q.sort();
cout << q; // Изход: 1 1 2 2 3 5 8 10
```
